# EMA Profiling Tool

Minimal profiling tool for EMA framework.

## Build

### Prerequisites

1. Make.
2. Gcc.
3. EMA Installed.
4. `EMA_INSTALL_DIR` environment variable setup and pointing to the EMA's
   installation location.

### Steps

1. Run `make` from this directory.

## Usage

1. `chmod +x ./profile` (if needed).
2. `./profiler --help` - to display a detailed help message.
